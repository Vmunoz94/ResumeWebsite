Experience:

Tank Wars
Tank Wars is a 2D player vs player tank game developed in Java using NetBeans IDE. It was a two-person project, which my partner and I had to communicate efficiently and effectively in order to meet the deadline. We split the project into multiple parts.
My tasks included:
  • Creating the game engine
  • Keyboard input
  • Tanks/Bullets and their movements
  • Mini-map/Splitscreen
  • Health/Lives system
  • Sound
  • Powerup
  • Game-Over screen
Galactic Mail
Galactic Mail is the second game my partner and I created using the same programming language and IDE. It is a single player game where the goal is to get the highest score by landing on each moon as quickly as possible. The goal for these projects was to practice good object-oriented programming in order to reuse some of the code to make the developing this game much easier.
My tasks for this game included:
  • Creating the scoring system
  • High score chart
  • Reply functionality
  • Level system
