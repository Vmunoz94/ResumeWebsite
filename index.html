Experience:

Tank Wars
Tank Wars was a two-person project, which my partner and I had to communicate efficiently and effectively in order to meet the deadline. The purpose of the project was to practice good object-oriented programming so that we can reuse some of the code to make developing the second game easier. We split the project into multiple parts, my tasks included; creating the game engine, tanks, bullets, mini-map, health system, lives counter, sound, powerup, and game over screen.
Galactic Mail
Galactic Mail was the second game we created. We were able to use a lot of the same classes as the tank game, which made developing this project much easier and faster. For this project I was tasked with creating the scoring system, high score chart, reset button, and level system.
